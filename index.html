Vehicle-to-Load (V2L) System Simulation

A full SPICE model of a bidirectional EV power system — AC grid charging via a diode-bridge rectifier and DC-to-AC inversion through an H-bridge PWM inverter. Outputs grid-quality 230 V / 50 Hz or 120 V / 60 Hz at the flick of a mode parameter, verified with a Python post-processing pipeline and rendered waveform plots.

DC-DC Power Converter Simulations

LTspice simulation library covering five switching converter topologies — Buck, Boost, Ćuk, and their synchronous variants — modelled with real component datasheets (IRFZ44N MOSFETs, Schottky diodes). Each topology accounts for conduction, switching, and gate-drive losses, making this a practical reference for power-supply design and topology comparison.

Analy — Email & Calendar Analytics

Full-stack email analytics product split across two repos. The Electron desktop app provides a Chart.js dashboard with Google OAuth2, local SQLite storage, and on-device AI analysis of your inbox. The Fastify/TypeScript backend handles Gmail sync, Calendar API integration, and exposes REST endpoints for sender trends, daily volume, and hourly usage patterns — deployed via Docker with GitHub Actions CI.
